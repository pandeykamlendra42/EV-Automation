# EV Automation (UBER) - Android | Kotlin | Firestore

Android apps for electrical vehicle automation that has the same functionality as the UBER app. There are two apps for customers and drivers, in this video, we are demonstrating the one complete cycle from cab booking to ride review after payment.

###Technology :
   - Apps -- Kotlin, Firestore
   - Backend - Python, Django
